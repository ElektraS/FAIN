Exemple d'utilisation : 

> ./main dans un terminal : créé une image noire 400*400

> clic droit pour ajouter des points

> touche 'c' pour fermer la courbe

> touche 'f' pour utiliser scanline pour remplir le polygone


/------------------------------------\
              INTERFACE
\------------------------------------/

> touche 'a' : active le mode APPEND

> touche 'v' : active le mode VERTEX

> touche 'e' : active le mode EDGE

> touche 'f' pour activer scanline pour remplir le polygone

> touche ECHAP ou 'q' : sortie de programme
